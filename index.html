Docushield offers a seamless and secure user experience

All your documents and images can be stored forever using the Marmalade NFT standards on Kadena. Three easy steps to use Docushield -

  • Account Creation
  • Document Upload.
  • Data Distribution & Retrieval

An NFT will be generated to act as an access key for the secured document uploaded by the user and stored in the user app/wallet using ACL access control methods for document retrieval.
